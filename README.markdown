# 🇺🇸 English

## Overview
Minecraft Clone is a desktop sandbox game inspired by Minecraft, developed with Python and PyQt6. The project includes world generation, block placement and destruction, inventory management, multiple languages, theme switching, player movement, day/night cycle, health and hunger systems, and an interactive graphical user interface.

## Features

- Procedural world generation
- Block placement and destruction
- Inventory system
- Health and hunger system
- Day and night cycle
- Multiple block types
- First-person camera controls
- Theme switching (Dark / Light)
- Multi-language support:
  - English
  - Persian (فارسی)
  - Chinese (中文)
- Pause menu
- World seed generation
- Adjustable world size
- FPS display
- Player statistics panel

## Requirements

- Python 3.10 or newer
- PyQt6

## Installation

### Install required libraries

```bash
pip install PyQt6
```

### Verify installation

```bash
python -c "import PyQt6; print('PyQt6 Installed Successfully')"
```

## Run the Project

```bash
python "Ursina Engine Minecraft Clone.py"
```

## Controls

| Key | Action |
|------|----------|
| W A S D | Move |
| Mouse | Look Around |
| Space | Jump |
| Left Click | Break Block |
| Right Click | Place Block |

| E | Inventory |
| ESC | Pause Menu |

## 项目简介

Minecraft Clone 是一个受 Minecraft 启发的桌面沙盒游戏，使用 Python 和 PyQt6 开发。项目包含世界生成、方块放置与破坏、背包系统、多语言支持、主题切换、玩家移动、昼夜循环以及交互式图形界面。

## 功能特点

- 程序化世界生成
- 方块放置与破坏
- 背包系统
- 生命值与饥饿系统
- 昼夜循环
- 多种方块类型
- 第一人称视角
- 深色与浅色主题
- 多语言支持
  - English
  - فارسی
  - 中文
- 暂停菜单
- 世界种子生成
- 可调节世界大小
- FPS 显示
- 玩家状态信息

## 环境要求

- PyQt6
